## What This Project Does

finESS is a **local Next.js + Prisma/SQLite uncertainty workbench**. A single operator on their own machine builds probabilistic models in one of five user-facing modes:

1. **Semantic Mode** (Phases A + B + D — complete): multi-turn conversation → clarifying questions → component identification → research via 7 mechanisms → distribution parameters with full citation chains → engine run → calibrated result.
2. **Forecast Mode**: time-series CSV → real Python ensemble sidecar → per-model weights + 95% interval + calibration feedback loop (R6-06).
3. **Real Data Mode**: CSV → empirical stats + optional LLM narration.
4. **Multi-LLM Proposer** (Path A — gated): parallel fan-out to all configured LLMs.
5. **Path A Simulation** (legacy / gated): single-LLM uncertainty graph + Monte Carlo.

**Tech Stack:** TypeScript (Next.js 14.2.35 App Router), React, Prisma 6.19 / SQLite, Python 3.11 / FastAPI sidecar, OpenRouter for LLMs, Tavily for web search (B2), `@xenova/transformers` BGE + `@lancedb/lancedb` for RAG (B3), Jest for tests, Docker Compose for the sidecar.
